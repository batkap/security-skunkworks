# Security Policy

This repository uses Security Skunkworks for local-first security review, requirement extraction, and remediation planning.

## Reporting

Use the repository issue tracker or maintainer contact path to report security concerns.

## Security Run State

The latest durable security run artifacts live under `.security-skunkworks/runs/${run_id}/`.

## Scope

This repository tracks threat models, security requirements, compliance mappings, and review outputs under `.security-skunkworks/` by default, and under `docs/security/` only when an explicit write mode allows canonical doc updates.
